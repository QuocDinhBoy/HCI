# 🎵 Thư mục nhạc theo cảm xúc

Đặt các file nhạc MP3 vào thư mục này với tên file chính xác như bên dưới:

| Cảm xúc | Tên file | Gợi ý loại nhạc |
|---|---|---|
| Bình thường (Neutral) | `neutral.mp3` | Nhạc ambient nhẹ nhàng, lo-fi |
| Vui vẻ (Happy) | `happy.mp3` | Nhạc sôi động, vui tươi |
| Buồn bã (Sad) | `sad.mp3` | Nhạc nhẹ nhàng, piano chậm |
| Tức giận (Angry) | `angry.mp3` | Nhạc mạnh, rock nhẹ |
| Sợ hãi (Fearful) | `fearful.mp3` | Nhạc thư giãn, an ủi |
| Ghê tởm (Disgusted) | `disgusted.mp3` | Nhạc chuyển đổi tâm trạng |
| Ngạc nhiên (Surprised) | `surprised.mp3` | Nhạc sáng tạo, phấn khích |

## Lưu ý
- Định dạng: **MP3** 
- Tên file phải đúng chính xác (lowercase, không dấu)
- Nên chọn nhạc phù hợp cho trẻ em
